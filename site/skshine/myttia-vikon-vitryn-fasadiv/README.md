# SK SHINE: миття вікон, вітрин і фасадів

Службова папка для погодження мʼякої SEO/AI-правки сторінки SK SHINE.

## Файли

- `color-changes.html` — версія для погодження: поточна сторінка з кольоровими позначками змінених фрагментів.
- `index.html` — чиста оновлена text-only версія без позначок.
- `source-live.html` — копія сторінки до правок.
- `changes-visible.html` — службова карта “було/стало” та SEO/AI пояснення.
