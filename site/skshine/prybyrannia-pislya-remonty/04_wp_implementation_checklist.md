# WordPress implementation checklist

Страница: https://skshine.com.ua/prybyrannia-pislya-remonty/
WP page ID: 3435

## Перед переносом

- [ ] Дождаться согласования собственника.
- [ ] Не менять URL.
- [ ] Не менять цены в карточках.
- [ ] Не менять фото и визуальный порядок блоков без отдельного решения.

## Rank Math

- [ ] Title: `Прибирання після ремонту у Хмельницькому | SK SHINE`.
- [ ] Description: `Прибирання після ремонту у Хмельницькому: від 65 грн/м², будівельний пил, сліди клею/фарби, кухня, санвузол, підлога, швидкий розрахунок.`
- [ ] Focus keyword: `прибирання після ремонту у хмельницькому`.
- [ ] Secondary phrase in content: `прибирання після ремонту Хмельницький`.

## Контент

- [ ] H1.
- [ ] Підзаголовок першого екрана.
- [ ] Текст форми розрахунку.
- [ ] Пояснення до цін.
- [ ] CTA після галереї.
- [ ] Блок `Як це працює`.

- [ ] FAQ: післяремонтний пил, ціна, відмінність від генерального, вікна, сміття, меблі, присутність, час, райони.

- [ ] Додати Service schema з цінами від 75/70/65 грн/м².
