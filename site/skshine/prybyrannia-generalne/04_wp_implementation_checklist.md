# Чеклист внедрения в WordPress

Страница: https://skshine.com.ua/prybyrannia-generalne/

## До публикации

- [ ] Проверить, что собственник согласовал текст.
- [ ] Сохранить текущую версию страницы/шаблона.
- [ ] Не менять URL.
- [ ] Не менять цены без отдельного подтверждения.
- [ ] Не удалять текущие фото и форму заявки.

## Что внести

- [ ] Title в Rank Math.
- [ ] Description в Rank Math.
- [ ] Focus keyword: `генеральне прибирання хмельницький`.
- [ ] H1.
- [ ] Подзаголовок первого экрана.
- [ ] Текст под формой расчёта.
- [ ] Пояснение к цене.
- [ ] CTA после галереи.
- [ ] Подписи блока `Як це працює`.
- [ ] Тексты блока `Чому обирають саме нас`.
- [ ] FAQ.
- [ ] Нижний SEO-текст.
- [ ] Alt для основных фото.
- [ ] FAQPage schema, если FAQ видим на странице.

## После публикации

- [ ] Проверить страницу на десктопе.
- [ ] Проверить страницу на мобильном.
- [ ] Проверить телефон, Viber, Telegram, форму.
- [ ] Проверить, что нет лишней подсветки из прототипа.
- [ ] Отправить URL на переобход в Google Search Console.
- [ ] Поставить дату публикации в очереди.
- [ ] Назначить проверку через 14 и 28 дней.
