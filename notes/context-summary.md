# Короткая памятка по проекту

## Цель

Собрать понятную структуру сайта по электротранспорту без перегруза главной страницы.

## Базовая иерархия

1. Главная сайта
2. Ремонт электротранспорта
3. Типы устройств:
   - электросамокаты
   - электроскутеры
   - электровелосипеды
   - гироскутеры
   - гироборды
   - моноколеса
4. Ветка электросамокатов:
   - бренды
   - поломки

## Почему именно так

1. Главная сайта должна оставаться широкой и сервисной
2. Страница `электротранспорт` работает как хаб
3. Страница `электросамокаты` — главная SEO и коммерческая посадочная
4. Бренды и поломки не выносятся на верхний уровень, чтобы не делать кашу
